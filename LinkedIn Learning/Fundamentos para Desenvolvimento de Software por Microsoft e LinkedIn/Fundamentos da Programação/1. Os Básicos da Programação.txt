Código interpretado tem as seguintes desvantagens, qualquer pessoa que tiver o interpretador consegue rodar o código, o código fonte é público, o código pode rodar mais lentamente pois ele tem de ser interpretado cada vez que ele roda. O fato do código fonte precisar ser interpretado separadamente para cada tipo de CPU, não é uma desvantagem, mas sim o que possibilita ele rodar em diferentes máquinas.
