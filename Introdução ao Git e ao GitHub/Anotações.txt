O Git Bash é um terminal extendido para otimizar o uso do Git.
-cd /
-dir = lista
-mkdir = cria pasta
-del/rmdir = apaga pasta
echo = imprime uma saída
ls = lista pastas e arquivos
ls -a = lista pastas e arquivos ocultos
git init = cria-se um repositório
git add = adiciona uma mudança no diretorio
git commit = cria um snapshot
