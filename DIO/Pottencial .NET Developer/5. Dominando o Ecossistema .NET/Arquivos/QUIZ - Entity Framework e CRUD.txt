O Entity Framework é muito usado em diversos projetos, pois auxilia muito o desenvolvedor na construção de uma aplicação. Dentre suas principais vantagens, podemos citar fácil integração com o banco de dados, como criação de tabelas e execução de comandos CRUD no banco de dados, sem a necessidade de escrevermos comandos SQL.
Ao obtermos um recurso de uma API, podemos passar algumas informações, dentre elas, o ID. Caso determinado ID não exista, então a API nos retornará um resultado HTTP, que é 404 - Not Found.
Para apagarmos um novo recurso de uma API, ou seja, remover um registro existente, usamos o verbo HTTP DELETE.
Para atualizarmos um recurso de uma API, ou seja, atualizar um registro existente, usamos o verbo HTTP PUT.
Para obtermos um recurso, ou seja, um determinado registro de uma API, usamos o verbo HTTP GET.
Qual o significado de CRUD? Create, Read, Update, Delete.
Para inserirmos um novo recurso de uma API, ou seja, incluir um novo registro, usamos o verbo HTTP POST.
